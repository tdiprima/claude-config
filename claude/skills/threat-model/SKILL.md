---
description: Create a quick threat model for a tool or system
---

Analyze the provided system or code and create a simple threat model.

Steps:

1. Identify system components.
2. Identify trust boundaries.
3. Identify external inputs.

Analyze risks including:

- injection
- privilege escalation
- secrets exposure
- unsafe configuration
- data leakage

Output:

1. System overview
2. Attack surface
3. Key risks
4. Recommended mitigations
